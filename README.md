Releitura simples e retrô do jogo de dinossauro da Google. :)

MODO DE USO:

1. Certifique-se de ter o VScode ou Git instalado em seu computador.
2. Clone este repositório e abra-o no editor (VScode).
3. Abra a pasta onde o repositório foi adicionado.
4. Procure o arquivo "index.html".
5. Abra-o e siga as instruções do jogo.
